## この節でできるようになること

- 損益通算できる4所得と、対象外・制限部分を説明できる。
- 医療費控除、社会保険料控除、iDeCo、生命保険料控除、基礎控除を正しい順序で計算できる。
- 連続ケースから所得控除合計と課税所得金額を求められる。

## 1. 損益通算

損益通算は、一定の所得で生じた損失を他の所得から差し引く制度です。対象となる基本4所得は次のとおりです。

- 不動産所得
- 事業所得
- 山林所得
- 譲渡所得

覚え方は「ふ・じ・さん・じょう」ですが、名称を覚えるだけでは不十分です。株式等や土地建物の譲渡損失、不動産所得の土地取得借入金利子部分、生活に通常必要でない資産の譲渡損失などには制限があります。

配当、給与、一時、雑所得の赤字は、原則として他所得と損益通算できません。

## 2. 損益通算の順序

FP3級では複雑な順序計算より、対象か否かの判断が中心です。解答時は次の順で確認します。

1. 赤字がどの所得区分で生じたか。
2. その所得に個別の通算制限がないか。
3. 通算できる金額だけを総合課税の所得から差し引く。
4. 退職所得・山林所得等を含む問題は、問題文の指示と法定順序を確認する。

## 3. 所得控除と税額控除は別

所得控除は、人的事情や政策目的を反映し、所得から差し引きます。税額控除は、課税所得に税率を掛けた後の税額から差し引きます。

| 所得控除 | 税額控除 |
| :--- | :--- |
| 基礎控除、医療費控除、社会保険料控除、生命保険料控除、扶養控除、iDeCo等 | 配当控除、住宅ローン控除など |

## 4. 基礎控除

2026年4月1日法令基準で使用する令和7年分・令和8年分の基礎控除は次のとおりです。

| 合計所得金額 | 基礎控除額 |
| :--- | :--- |
| 132万円以下 | 95万円 |
| 132万円超336万円以下 | 88万円 |
| 336万円超489万円以下 | 68万円 |
| 489万円超655万円以下 | 63万円 |
| 655万円超2,350万円以下 | 58万円 |
| 2,350万円超2,400万円以下 | 48万円 |
| 2,400万円超2,450万円以下 | 32万円 |
| 2,450万円超2,500万円以下 | 16万円 |
| 2,500万円超 | 0円 |

令和8年度税制改正による追加の引上げは2026年12月1日施行であり、2026年4月1日法令基準の試験には使用しません。

## 5. 医療費控除

> 医療費控除 ＝ 支払医療費－保険金等の補填額－10万円

総所得金額等が200万円未満なら、10万円ではなく総所得金額等の5％を差し引きます。控除上限は200万円です。補填額は、その給付の対象となった医療費を限度に差し引き、余剰を他の医療費から差し引きません。

### 例

支払医療費30万円、入院給付金5万円、総所得200万円以上なら、30万円－5万円－10万円＝**15万円**です。

## 6. その他の主要控除

- 社会保険料控除：本人または生計を一にする親族のため支払った対象保険料の全額。
- 小規模企業共済等掛金控除：iDeCo掛金等の全額。
- 生命保険料控除：新契約では一般、介護医療、個人年金の各区分で計算し、所得税の合計上限は12万円。
- 扶養控除：原則16歳以上の扶養親族。所得要件と年齢区分を確認する。
- 配偶者控除・配偶者特別控除：納税者本人の合計所得金額が1,000万円を超える場合は適用なし。

## 7. 連続ケースの所得控除

青葉直人さんの合計所得金額は451万円です。次の控除があるものとします。

- 社会保険料控除：70万円
- 医療費控除：15万円
- iDeCo：24万円
- 生命保険料控除：8万円（計算済み）
- 基礎控除：68万円

所得控除合計は70＋15＋24＋8＋68＝**185万円**です。

課税所得金額は451万円－185万円＝**266万円**です。1,000円未満は切り捨てます。

## 8. よくある誤り

- 医療費30万円から補填額を引かずに20万円控除する。
- iDeCo掛金を税額から直接引く。
- 合計所得451万円なのに基礎控除58万円を選ぶ。
- 不動産所得△30万円を全額通算し、土地取得借入金利子10万円を除外しない。
- 「副業所得20万円以下は非課税」と誤解する。申告不要制度と所得の非課税は別です。

## 9. 自己点検チェック

1. 損失の所得区分と制限部分を確認したか。
2. 所得控除を税率適用前に引いたか。
3. 基礎控除は合計所得金額の段階を使ったか。
4. 課税所得の1,000円未満を切り捨てたか。
