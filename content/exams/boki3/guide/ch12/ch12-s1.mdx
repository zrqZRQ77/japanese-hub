## この節の目標

この節では、試算表の名前を覚えるだけでなく、次のことができる状態を目指します。

- 総勘定元帳の残高から残高試算表を作成できる
- 借方合計と貸方合計が一致する理由を説明できる
- 試算表で発見しやすい誤りと、発見できない誤りを区別できる
- 合計が一致しないときに、確認する順番を決めて調べられる

> 対応範囲：日商簿記3級・2026年度出題区分表
> 最終確認日：2026年7月17日

## 試算表とは

**試算表（しさんひょう：Trial Balance / T/B）**は、総勘定元帳に記録された各勘定の金額を集計し、仕訳や転記に大きな誤りがないかを確認するための表です。

帳簿の流れは次のとおりです。

**取引 → 仕訳帳 → 総勘定元帳 → 試算表 → 決算整理 → 財務諸表**

仕訳では、1つの取引について借方と貸方に同額を記入します。そのため、正しく記帳・転記・集計されていれば、試算表でも借方合計と貸方合計は一致します。これを**貸借平均の原理**といいます。

ただし、合計が一致したからといって、すべての記録が正しいとは限りません。試算表は「誤りを発見するための重要な確認表」ですが、「正しさを完全に保証する表」ではありません。

## 試算表の3種類

| 種類 | 集計する金額 | 主な使い方 |
|---|---|---|
| 合計試算表 | 各勘定の借方合計・貸方合計 | 転記された取引量や片側転記の確認 |
| 残高試算表 | 各勘定の借方残高または貸方残高 | 決算整理や財務諸表作成の基礎 |
| 合計残高試算表 | 合計と残高の両方 | 合計と残高を同時に確認 |

日商簿記3級では、どの形式でも各勘定の性質と借貸を判断できることが必要です。特に決算問題では、残高試算表や決算整理前残高試算表が出発点になります。

## 総勘定元帳から残高試算表を作る

青葉商事株式会社で、次の5つの勘定残高が確認されたとします。

金額単位：円

| 勘定科目 | 借方合計 | 貸方合計 | 残高の計算 | 残高 |
|---|---:|---:|---|---:|
| 現金 | 180,000 | 60,000 | 180,000－60,000 | 借方120,000 |
| 売掛金 | 250,000 | 70,000 | 250,000－70,000 | 借方180,000 |
| 買掛金 | 40,000 | 190,000 | 190,000－40,000 | 貸方150,000 |
| 売上 | 20,000 | 500,000 | 500,000－20,000 | 貸方480,000 |
| 仕入 | 330,000 | 0 | 330,000－0 | 借方330,000 |

残高試算表には、借方合計・貸方合計をそのまま記入するのではなく、**差額である残高**を記入します。

| 勘定科目 | 借方残高 | 貸方残高 |
|---|---:|---:|
| 現金 | 120,000 |  |
| 売掛金 | 180,000 |  |
| 買掛金 |  | 150,000 |
| 売上 |  | 480,000 |
| 仕入 | 330,000 |  |
| 合計 | 630,000 | 630,000 |

### 作成手順

1. 各勘定の借方合計と貸方合計を確認する
2. 大きい側から小さい側を差し引いて残高を求める
3. 残高を大きい側に記入する
4. すべての勘定を集計する
5. 試算表の借方合計と貸方合計を照合する

## 試算表で発見しやすい誤り・発見できない誤り

| 誤りの例 | 借貸は一致するか | 試算表での発見 | 理由 |
|---|---|---|---|
| 借方だけを転記し、貸方を転記しなかった | 一致しない | 発見しやすい | 片側だけが試算表に反映される |
| 借方100,000円、貸方10,000円と転記した | 一致しない | 発見しやすい | 左右の金額が異なる |
| 合計を計算し間違えた | 一致しないことが多い | 発見しやすい | 一方の集計額が変わる |
| 取引そのものを仕訳しなかった | 一致することがある | 発見できない | 借方・貸方の両方が記録されない |
| 借方と貸方を完全に逆にした | 一致する | 発見できない | 左右の金額自体は同額 |
| 借方・貸方とも同じ誤った金額にした | 一致する | 発見できない | 両側が同額のまま誤っている |
| 正しい金額を別の勘定科目に転記した | 一致することがある | 発見できない | 試算表全体の左右合計は変わらない |

重要なのは、**取引全体の記帳漏れ**と**一方だけの転記漏れ**を区別することです。

- 取引全体を記録していない：借方・貸方の両方がないため、合計は一致することがある
- 借方または貸方の片側だけ転記していない：左右差が生じるため、合計は一致しない

## 合計が一致しないときの確認順序

試算表が不一致のとき、無作為に帳簿を見直すのではなく、次の順番で確認します。

1. **試算表の再集計**：縦の合計をもう一度計算する
2. **残高の再計算**：各勘定で借方合計と貸方合計の差額を確認する
3. **片側転記の確認**：仕訳帳の借方・貸方に照合印を付ける
4. **金額の確認**：桁違い、数字の入れ替え、転記額の違いを調べる
5. **借貸方向の確認**：借方残高を貸方に書いていないか確認する
6. **勘定科目の確認**：合計が一致した後も、誤った科目への転記がないか確認する

### 自己チェック

次の誤りは、試算表の借方合計と貸方合計が一致していても発見できない可能性があります。

- 商品100,000円の掛仕入を、借方・貸方とも10,000円で記録した
- 売掛金の回収を、借方と貸方を完全に逆にした
- 取引を仕訳帳にまったく記録しなかった

一方、仕訳は正しいのに貸方だけ総勘定元帳へ転記しなかった場合は、通常、試算表は不一致になります。
