## この節の目標

精算表を完成しただけでは、決算は終わりません。この節では、精算表で確定した金額が財務諸表と帳簿の締切へどうつながるかを学びます。

- 精算表と正式な財務諸表の違いを説明できる
- 収益・費用を損益勘定へ振り替えられる
- 当期純利益または純損失を繰越利益剰余金へ振り替えられる
- 資産・負債・純資産と収益・費用で締切方法が異なる理由を説明できる

> 対応範囲：日商簿記3級・2026年度出題区分表
> 最終確認日：2026年7月17日

## 精算表の後に行うこと

決算の全体像は次のとおりです。

**決算整理前残高試算表**
→ **決算整理仕訳**
→ **精算表・決算整理後残高の確認**
→ **損益計算書と貸借対照表の作成**
→ **収益・費用の損益勘定への振替**
→ **純損益の繰越利益剰余金への振替**
→ **帳簿の締切と次期繰越**

精算表は、決算整理と財務諸表作成を助ける**作業表**です。精算表を作成しても、それだけで総勘定元帳の残高が変わるわけではありません。決算整理仕訳や決算振替仕訳は、実際に仕訳帳と総勘定元帳へ記帳する必要があります。

## 1. 精算表から財務諸表を作る

[12-2の青葉商事株式会社の事例](/exams/boki3/guide/ch12?section=ch12-s2)では、当期純利益は157,000円でした。

精算表の損益計算書欄から、収益と費用を正式な損益計算書へ移します。貸借対照表欄から、決算日現在の資産・負債・純資産を正式な貸借対照表へ移します。

| 精算表の欄 | 表す内容 | 正式な書類 |
|---|---|---|
| 損益計算書欄 | 1年間の収益・費用と経営成績 | 損益計算書 |
| 貸借対照表欄 | 決算日現在の資産・負債・純資産 | 貸借対照表 |

財務諸表の表示形式は[第13章 財務諸表](/exams/boki3/guide/ch13)で詳しく確認します。

## 2. 収益を損益勘定へ振り替える

収益勘定は通常、貸方残高です。貸方残高をゼロにするには、同額を借方へ記入し、相手科目を損益とします。

青葉商事の売上は3,200,000円です。

| 借方科目 | 借方金額 | 貸方科目 | 貸方金額 |
|---|---:|---|---:|
| 売上 | 3,200,000 | 損益 | 3,200,000 |

この仕訳により、売上勘定の残高はゼロになり、損益勘定の貸方へ収益3,200,000円が集まります。

## 3. 費用を損益勘定へ振り替える

費用勘定は通常、借方残高です。借方残高をゼロにするには、同額を貸方へ記入し、借方を損益とします。

青葉商事の決算整理後の費用合計は3,043,000円です。

| 借方科目 | 借方金額 | 貸方科目 | 貸方金額 |
|---|---:|---|---:|
| 損益 | 3,043,000 | 仕入 | 1,760,000 |
|  |  | 給料 | 750,000 |
|  |  | 消耗品費 | 60,000 |
|  |  | 保険料 | 80,000 |
|  |  | 支払利息 | 30,000 |
|  |  | 支払家賃 | 240,000 |
|  |  | 貸倒引当金繰入 | 3,000 |
|  |  | 減価償却費 | 120,000 |

これで各費用勘定の残高はゼロになり、損益勘定の借方へ費用3,043,000円が集まります。

## 4. 損益勘定で当期純利益を確認する

損益勘定には次の金額が集まりました。

| 損益勘定の借方 | 金額 | 損益勘定の貸方 | 金額 |
|---|---:|---|---:|
| 費用合計 | 3,043,000 | 収益合計 | 3,200,000 |
| 当期純利益 | 157,000 |  |  |
| 合計 | 3,200,000 | 合計 | 3,200,000 |

貸方の収益が借方の費用より157,000円多いため、損益勘定は貸方残高157,000円です。この金額は、精算表と損益計算書で求めた当期純利益と一致します。

### 三つの金額が一致する

1. 精算表の当期純利益
2. 損益計算書の当期純利益
3. 損益勘定の差額

この三つが一致しなければ、整理仕訳、分類、集計または振替のどこかに誤りがあります。

## 5. 当期純利益を繰越利益剰余金へ振り替える

株式会社では、当期純利益を純資産の**繰越利益剰余金**へ振り替えます。

| 借方科目 | 借方金額 | 貸方科目 | 貸方金額 |
|---|---:|---|---:|
| 損益 | 157,000 | 繰越利益剰余金 | 157,000 |

この仕訳により、損益勘定もゼロになり、当期に獲得した利益が純資産に加算されます。

青葉商事の決算整理前の繰越利益剰余金は381,000円なので、他に配当等がないという前提では、振替後は次の金額になります。

381,000＋157,000＝538,000円

### 当期純損失の場合

損失の場合は、繰越利益剰余金を減少させます。

| 借方科目 | 借方金額 | 貸方科目 | 貸方金額 |
|---|---:|---|---:|
| 繰越利益剰余金 | ××× | 損益 | ××× |

## 6. 資産・負債・純資産を締め切る

収益・費用は1年間の成績を表すため、決算振替によって残高をゼロにします。一方、資産・負債・純資産は決算日の財政状態を表し、翌期へ引き継ぐ必要があります。

| 勘定の種類 | 決算時の処理 | 翌期への扱い |
|---|---|---|
| 収益・費用 | 損益勘定へ振り替えて残高をゼロにする | 翌期はゼロから開始 |
| 資産・負債・純資産 | 残高を次期繰越として締め切る | 前期繰越として引き継ぐ |

例えば、現金勘定の借方残高が260,000円なら、貸方に「次期繰越260,000円」を記入して左右を一致させます。翌期首には借方へ「前期繰越260,000円」と記入します。

「帳簿をリセットする」という表現だけでは不正確です。正しくは、**収益・費用はゼロにし、資産・負債・純資産は残高を翌期へ繰り越す**処理です。

## よくある誤り

| 誤り | なぜ誤りか |
|---|---|
| 精算表を作ったので整理仕訳の記帳は不要と考える | 精算表は作業表であり、元帳の残高を直接変更しない |
| 売上を貸方のまま損益へ振り替える | 売上をゼロにするため、売上は借方へ記入する |
| 費用を借方のまま損益へ振り替える | 費用をゼロにするため、各費用は貸方へ記入する |
| 当期純利益を資本金へ振り替える | 現行の日商簿記3級は株式会社を前提とし、繰越利益剰余金へ振り替える |
| 現金や買掛金も損益勘定へ振り替える | 資産・負債は期間損益ではなく、翌期へ残高を繰り越す |
| 精算表の利益と損益勘定の差額を別々に計算する | 同じ収益・費用から求めるため、本来は同額になる |

## 決算の最終チェック

- 決算整理仕訳を仕訳帳と総勘定元帳へ記帳したか
- 精算表、損益計算書、損益勘定の純損益が一致しているか
- 収益・費用の残高がゼロになったか
- 純損益を繰越利益剰余金へ振り替えたか
- 資産・負債・純資産を次期へ繰り越したか
- 貸借対照表の借方合計と貸方合計が一致しているか
