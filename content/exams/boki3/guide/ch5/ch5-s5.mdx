## 補助簿を使う目的

仕訳帳と総勘定元帳だけでも会社全体の金額は分かりますが、「どの商品が何個残っているか」「どの仕入先から買ったか」「どの得意先へ売ったか」までは十分に追えません。そこで、取引の内訳を補助簿で管理します。

| 補助簿 | 管理する内容 |
|---|---|
| 仕入帳 | 商品仕入の明細、仕入返品など |
| 売上帳 | 商品売上の明細、売上返品など |
| 商品有高帳 | 商品別の受入・払出・残高を原価で管理 |
| 売掛金元帳 | 得意先別の売掛金残高 |
| 買掛金元帳 | 仕入先別の買掛金残高 |

補助簿は仕訳の代わりではありません。掛売上なら、仕訳帳・総勘定元帳・売上帳・売掛金元帳・商品有高帳がそれぞれ異なる目的で動きます。

## 商品有高帳の最重要ルール

商品有高帳は、商品の種類ごとに次の3欄を記録します。

- **受入**：仕入や売上返品によって商品が増えた
- **払出**：売上や仕入返品によって商品が減った
- **残高**：取引後に手元へ残っている数量と原価

> 商品有高帳は、売価ではなく原価で記入します。

販売時の売上仕訳が1個2,000円でも、商品有高帳の払出単価が1,120円なら、払出欄には1,120円を使います。

## 同じ資料を2つの方法で計算する

商品Aについて、4月の取引は次のとおりです。平均単価に円未満の端数は生じません。

| 日付 | 取引 | 数量 | 単価・売価 |
|---|---|---:|---:|
| 4/1 | 前月繰越 | 40個 | 原価1,000円 |
| 4/5 | 仕入 | 60個 | 原価1,200円 |
| 4/10 | 売上 | 50個 | 売価1,800円 |
| 4/18 | 仕入 | 50個 | 原価1,360円 |
| 4/25 | 売上 | 60個 | 売価2,000円 |

受入原価の合計は次のとおりです。

> 40,000円 ＋ 72,000円 ＋ 68,000円 ＝ 180,000円

## 先入先出法

先入先出法は、先に受け入れた商品から先に払い出したと仮定する方法です。単価の異なる在庫が残る場合は、残高を単価別に分けて管理します。

### 先入先出法の商品有高帳

| 日付 | 摘要 | 受入数量 | 受入単価 | 受入金額 | 払出数量 | 払出単価 | 払出金額 | 残高 |
|---|---|---:|---:|---:|---:|---:|---:|---|
| 4/1 | 前月繰越 | 40 | 1,000 | 40,000 |  |  |  | 40個・40,000円 |
| 4/5 | 仕入 | 60 | 1,200 | 72,000 |  |  |  | 100個・112,000円（※1） |
| 4/10 | 売上 |  |  |  | 50 | 1,000・1,200（※2） | 52,000 | 50個・60,000円 |
| 4/18 | 仕入 | 50 | 1,360 | 68,000 |  |  |  | 100個・128,000円（※3） |
| 4/25 | 売上 |  |  |  | 60 | 1,200・1,360（※4） | 73,600 | 40個・54,400円 |

### 表中の注記と計算過程

**※1　4月5日の仕入後残高**

> 40個 × 1,000円 ＋ 60個 × 1,200円 ＝ 112,000円

**※2　4月10日の払出**

古い在庫から40個を払い出し、足りない10個を次の単価から払い出します。

> 40個 × 1,000円 ＋ 10個 × 1,200円 ＝ 52,000円

払出後は、単価1,200円の商品が50個、60,000円残ります。

**※3　4月18日の仕入後残高**

> 50個 × 1,200円 ＋ 50個 × 1,360円 ＝ 128,000円

**※4　4月25日の払出**

古い1,200円の商品50個をすべて払い出し、1,360円の商品を10個払い出します。

> 50個 × 1,200円 ＋ 10個 × 1,360円 ＝ 73,600円

払出後は、単価1,360円の商品が40個、54,400円残ります。

売上原価は払出金額の合計です。

> 52,000円 ＋ 73,600円 ＝ 125,600円

## 移動平均法

移動平均法は、商品を仕入れるたびに、在庫全体の平均単価を計算し直す方法です。

> 新平均単価 ＝（仕入直前の残高金額＋今回の仕入金額）÷（仕入直前の残高数量＋今回の仕入数量）

### 4月5日の新平均単価

> （40,000円 ＋ 72,000円）÷（40個 ＋ 60個）＝ 1,120円

4月10日の払出額：

> 50個 × 1,120円 ＝ 56,000円

払出後の残高は50個、56,000円です。

### 4月18日の新平均単価

> （56,000円 ＋ 68,000円）÷（50個 ＋ 50個）＝ 1,240円

4月25日の払出額：

> 60個 × 1,240円 ＝ 74,400円

期末残高は40個、49,600円です。

### 移動平均法の商品有高帳

| 日付 | 摘要 | 受入数量 | 受入単価 | 受入金額 | 払出数量 | 払出単価 | 払出金額 | 残高数量 | 残高単価 | 残高金額 |
|---|---|---:|---:|---:|---:|---:|---:|---:|---:|---:|
| 4/1 | 前月繰越 | 40 | 1,000 | 40,000 |  |  |  | 40 | 1,000 | 40,000 |
| 4/5 | 仕入 | 60 | 1,200 | 72,000 |  |  |  | 100 | 1,120 | 112,000 |
| 4/10 | 売上 |  |  |  | 50 | 1,120 | 56,000 | 50 | 1,120 | 56,000 |
| 4/18 | 仕入 | 50 | 1,360 | 68,000 |  |  |  | 100 | 1,240 | 124,000 |
| 4/25 | 売上 |  |  |  | 60 | 1,240 | 74,400 | 40 | 1,240 | 49,600 |

売上原価：

> 56,000円 ＋ 74,400円 ＝ 130,400円

## 売上総利益までつなげる

売上高は両方の方法で同じです。

- 4月10日：50個 × 1,800円 ＝ 90,000円
- 4月25日：60個 × 2,000円 ＝ 120,000円
- 売上高合計：210,000円

| 方法 | 売上高 | 売上原価 | 売上総利益 | 期末商品 |
|---|---:|---:|---:|---:|
| 先入先出法 | 210,000 | 125,600 | 84,400 | 54,400 |
| 移動平均法 | 210,000 | 130,400 | 79,600 | 49,600 |

どちらも次の関係が成立します。

> 受入原価180,000円 ＝ 売上原価 ＋ 期末商品

先入先出法：125,600円＋54,400円＝180,000円

移動平均法：130,400円＋49,600円＝180,000円

## 返品と繰越の記入

- 仕入返品：商品が外へ出るため払出欄へ原価で記入
- 売上返品：商品が戻るため受入欄へ原価で記入
- 月末：残高を`次月繰越`として締め、翌月の受入欄へ`前月繰越`として移す

返品の単価は、問題文の指示と元の入出庫資料を確認します。売価を使用しないことが重要です。

## よくある誤り

| 誤り | 修正方法 |
|---|---|
| 売価を払出単価に使う | 商品有高帳は原価で記入する |
| 先入先出法で50個すべてを1,000円にする | 古い在庫を使い切ったら次の単価へ移る |
| 移動平均法で仕入後も古い平均単価を使う | 仕入のたびに平均を再計算する |
| 平均単価を単純平均する | 金額合計÷数量合計で加重平均する |
| 仕入諸掛りを受入原価から外す | 当店負担分を商品原価へ含める |
| 期末数量だけ確認して金額を検算しない | 受入原価＝払出原価＋期末原価を確認する |

## 3つの検算

1. 数量：前月繰越＋受入−払出＝期末数量
2. 金額：前月繰越原価＋仕入原価＝売上原価＋期末商品
3. 利益：売上高−売上原価＝売上総利益
