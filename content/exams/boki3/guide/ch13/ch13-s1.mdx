## 第13章は最終報告書を完成させる章

第1章・第2章で学んだ五要素の定義を繰り返すのではなく、第12章の精算表で確定した金額を、実際の損益計算書と貸借対照表へ移す手順を学びます。

統一ケースは青葉サービス株式会社です。会計期間は4月1日から3月31日まで、金額単位は円です。

## 調整後の収益・費用残高

決算整理後、精算表の損益計算書欄に次の残高が集計されました。

| 科目 | 区分 | 金額 |
|---|---|---:|
| 売上 | 収益 | 700,000 |
| 受取利息 | 収益 | 4,000 |
| 売上原価 | 費用 | 420,000 |
| 給料 | 費用 | 100,000 |
| 支払家賃 | 費用 | 48,000 |
| 水道光熱費 | 費用 | 20,000 |
| 減価償却費 | 費用 | 30,000 |
| 保険料 | 費用 | 6,000 |

精算表に入る前の売上・仕入残高ではなく、決算整理を反映した後の金額を使います。

## ステップ1：収益合計を計算する

> 売上700,000円＋受取利息4,000円＝704,000円

## ステップ2：費用合計を計算する

> 420,000円＋100,000円＋48,000円＋20,000円＋30,000円＋6,000円＝624,000円

## ステップ3：当期純利益を計算する

> 当期純利益＝収益704,000円−費用624,000円＝80,000円

収益が費用を上回るため、当期純利益です。費用が収益を上回れば当期純損失になります。

## 損益計算書

青葉サービス株式会社

自 20X1年4月1日　至 20X2年3月31日

金額単位：円

| 費用 | 金額 | 収益 | 金額 |
|---|---:|---|---:|
| 売上原価 | 420,000 | 売上 | 700,000 |
| 給料 | 100,000 | 受取利息 | 4,000 |
| 支払家賃 | 48,000 |  |  |
| 水道光熱費 | 20,000 |  |  |
| 減価償却費 | 30,000 |  |  |
| 保険料 | 6,000 |  |  |
| 当期純利益 | 80,000 |  |  |
| 合計 | 704,000 | 合計 | 704,000 |

当期純利益は左右を一致させるため費用側へ表示されますが、費用ではありません。収益と費用の差額です。

## 精算表から転記する順序

1. 損益計算書欄の貸方科目を収益として集める
2. 損益計算書欄の借方科目を費用として集める
3. 収益合計と費用合計を計算する
4. 差額を当期純利益または当期純損失とする
5. 当期純利益を費用側、当期純損失を収益側へ記入して合計を一致させる

## 売上原価を仕入残高のまま載せない

三分法では期中に`仕入`を使いますが、財務諸表には当期に販売した商品の原価である`売上原価`を表示します。

> 売上原価＝期首商品＋当期仕入純額−期末商品

このケースでは決算整理後の売上原価420,000円を使います。

## 期間を正しく表示する

損益計算書は「一定期間」の経営成績を示します。

- 正しい表示：自4月1日 至3月31日
- 貸借対照表のような「3月31日現在」ではない

## 収益・費用の漏れを確認する

### 収益

- 売上
- 受取利息
- 受取手数料
- 雑益など

### 費用

- 売上原価
- 給料
- 支払家賃
- 水道光熱費
- 減価償却費
- 保険料
- 支払利息など

精算表の損益計算書欄にある科目を一つずつ照合します。

## よくある誤り

### 誤り1：売上から現金入金額だけを載せる

売掛金による販売も売上に含まれます。入金額ではなく当期売上額を使います。

### 誤り2：期末商品を費用に含めたままにする

期末商品は翌期へ繰り越す資産です。売上原価から除きます。

### 誤り3：減価償却費をB/Sへ載せる

減価償却費はP/L、減価償却累計額はB/Sです。

### 誤り4：当期純利益を収益科目として扱う

当期純利益は収益と費用の差額であり、独立した収益科目ではありません。

### 誤り5：決算整理前の金額を使う

見越し・繰延べ、減価償却、売上原価などを反映した調整後残高を使います。

## 検算

- 収益合計704,000円
- 費用合計624,000円
- 当期純利益80,000円
- 費用624,000円＋利益80,000円＝収益704,000円
